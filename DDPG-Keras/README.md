# Installation Dependencies:

* Python 2.7
* Keras
* Tensorflow

# How to Run?

```
git clone https://github.com/theSparta/DDPG-Keras.git
cd DDPG-Keras
python ddpg.py 
```

(Change the flag **train_indicator**=1 in ddpg.py if you want to train the network)
